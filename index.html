Vad vi kommer göra

I denna kurs kommer du själv få lära dig skriva kod och bygga många olika typer av hemsidor för olika ändamål. Till en början kommer det vara relativt noga styrt kring hur du ska bygga hemsidorna. Längre fram kommer du att få mer frihet i hur dina hemsidor ska se ut.

Visual Studio Code

Vi arbetar främst med Visual Studio Code för att skriva HTML, CSS och Javascript. Visual Studio Code är ett stilren redigerare med kraftulla tillägg (extensions). Ladda ner Visual Studio Code här och installera om du inte har det på din dator än.

Några snabbkommandon

Här är några snabbkommandon du bör lära dig så du kan jobba mer effektivt:

  1. CTRL + Z = ångra
  2. CTRL + SHIFT + Z = gör om
  3. CTRL + ALT + ↓ / ↑ = Lägg till markör
  4. ESC = ta bort flera markörer
  5. CTRL + D = markera samma
  6. CTRL + SHIFT + L = hitta alla i dokumentet
  7. ALT + ↓ / ↑ = flytta aktuell rad upp eller ned
